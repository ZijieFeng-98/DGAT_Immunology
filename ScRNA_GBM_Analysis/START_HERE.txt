📋 QUICK START (NO INTERACTION NEEDED!)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  Install Python (ONE TIME ONLY)
   → Download: https://www.python.org/downloads/
   → Get: Python 3.10.x
   → ✓ CHECK "Add Python to PATH" when installing!
   → Install and restart your terminal


2️⃣  Run ONE Command (Does Everything!)
   
   Open PowerShell in this folder and run:
   
   .\setup_complete.ps1
   
   This will automatically:
   ✓ Create Python environment
   ✓ Install all packages (~15 min)
   ✓ Generate demo test data
   ✓ Verify everything works
   ✓ Show you next steps


3️⃣  Test It Works (5 minutes)
   
   After setup completes, test with demo data:
   
   python scripts\sc_pipeline.py --tumour_path data\raw\demo_tumour\ --normal_path data\raw\demo_normal\ --output_dir results\demo\ --max_genes 5000


4️⃣  Run Real Analysis
   
   Download real data (see DATA_DOWNLOAD_GUIDE.md)
   Then run interactive mode:
   
   python scripts\step_by_step.py

Problem: "Python not found"
Solution: Reinstall with "Add to PATH" checked, restart terminal

Problem: "Permission denied" on PowerShell
Solution: Run as Administrator or:
         Set-ExecutionPolicy -Scope CurrentUser RemoteSigned

Problem: Setup fails
Solution: Run manually:
         pip install -r requirements.txt

Problem: Need help
Solution: Read SETUP_COMPLETE.md for detailed guide
